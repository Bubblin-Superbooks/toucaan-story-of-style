Modern web browsers too have evolved and gotten better and we no longer need a heavy-handed reset to force consistency across vendors. In light of these tectonic shifts, Toucaan aims to move away from traditional reset or reboot css technique and implement a much simpler and lighter strategy instead. It is a tropical new CSS framework for the new web, with fewer defaults, newer patterns and a much simpler cascade. Welcome to Toucaan.
